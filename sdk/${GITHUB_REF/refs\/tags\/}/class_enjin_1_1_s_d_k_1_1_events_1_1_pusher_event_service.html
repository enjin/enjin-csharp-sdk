Task Enjin.SDK.Events.PusherEventService.Start ( )

Starts this service. See Start(Platform) to start this service with new platform details.

The task returned may complete exceptionally if the Platform property lacks the required Pusher data to connect. Ensure that Enjin.SDK.Shared.GetPlatform.WithNotificationDrivers is called on the Enjin.SDK.Shared.GetPlatform request before sending it to the platform.
