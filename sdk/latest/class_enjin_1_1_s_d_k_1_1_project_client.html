void Enjin.SDK.ProjectClient.Auth ( AccessToken?  accessToken)

Authenticates the client using the given access token model.

Parameters
accessTokenThe access token model.
Exceptions
ArgumentExceptionIf accessToken is not null and either its AccessToken.Token or AccessToken.ExpiresIn properties are null.

If this client has automatic reauthentication enabled, then this method may halt the reauthentication timer when accessToken is null. Otherwise the timer will be restarted.
